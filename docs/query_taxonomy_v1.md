# NCCU 課程搜尋 Query Taxonomy v1

## 設計原則
涵蓋政大學生搜課時所有可能的意圖、限制條件、語言風格。
每個 query 應「像學生真的會說出來的話」，不能直接抄課名。

---

## 1. 意圖分類（Intent）

### A. 主題型（Topic）— 最多，約 35%
學生知道自己想學什麼主題，不知道叫什麼課。
```
想學機器學習
有沒有教量子力學的課
我想了解國際法
想修跟品牌行銷有關的課
有沒有教怎麼打官司的
```

### B. 技能型（Skill）— 約 15%
想學特定技能或工具，不是學科知識。
```
想學 Python 程式
有沒有教 Excel/統計軟體
想學投資理財
想學簡報設計
有沒有教公務員考試的課
```

### C. 師型（Teacher）— 約 10%
知道特定老師名字，想找他開的課。
```
想修顧慕晴的課
李宗榮老師有開什麼課
有沒有推薦的某某教授的課
```

### D. 模糊/感受型（Vague/Feeling）— 約 15%
不知道具體想學什麼，描述感受或目的。
```
想找有趣的通識
好過的英文授課課
想找分數好拿的輕鬆課
有沒有很實用的選修課
想修個對未來工作有幫助的課
```

### E. 約束型（Constraint）— 約 20%
有明確的時段、語言、系所、學分等硬限制。

#### E1. 時段約束
```
週三下午有什麼好課
不要早八的課
下午兩點到五點有沒有空著
想找週一沒有排課的通識
```

#### E2. 語言約束
```
想找英文授課的課
中文授課，比較容易過的課
有沒有全英文的社科課
```

#### E3. 學分/必選修約束
```
想找三學分的選修
還需要補通識學分
想修2學分的體育課
```

#### E4. 系所/身分約束
```
外院能修的商院課
非本系能修的法律課
大一能修的進階課嗎
```

#### E5. 教室/地點約束
```
有沒有在百年樓上課的
想找有實體教室的課，不要線上
```

### F. 複合型（Compound）— 約 5%
同時有多個約束，最貼近真實複雜使用情境。
```
週二下午英文授課的通識，不要早八
商院大三選修，不要衝到週五
想找三學分、外院能修、跟資料科學有關的課
```

---

## 2. 語言風格維度（Style）

| 風格 | 說明 | 範例 |
|---|---|---|
| 正式 | 完整問句 | 「請問有沒有關於機器學習的課程？」|
| 口語 | 日常聊天語氣 | 「有沒有教 ML 的啊」|
| 疑問句 | 有沒有/嗎/？ | 「有沒有好的量化分析課啊？」|
| 陳述句 | 我想/我要 | 「我想找跟法律有關的課」|
| 簡短 | 4-8 字 | 「機器學習課程」|
| 長句 | 含多條件 | 「有沒有週三不用去、英文授課、外院可修的三學分選修」|

---

## 3. 否定/排除維度（Negation）

使用者明確說「不要」的限制：
```
不要早八
不想上週五的課
不要線上課
不要必修（只想選選修）
不要有期末考的課
不要需要報告的課
老師不要太嚴格
```

---

## 4. 精確度維度（Specificity）

| 精確度 | 說明 | 範例 |
|---|---|---|
| 精確 | 知道確切領域/老師 | 「財務報表分析」|
| 半精確 | 知道大方向 | 「跟財務有關的課」|
| 模糊 | 只有感受 | 「有趣的選修」|
| 錯誤拼寫 | 口語/手殘打錯 | 「機器學細」「機率統記」|

---

## 5. 生成矩陣（每種組合生成 2-3 筆）

| Intent | Style | Negation | Specificity |
|---|---|---|---|
| A主題 × E1時段 | 口語 | 不要早八 | 半精確 |
| B技能 × E2語言 | 疑問句 | 無 | 精確 |
| D模糊 × E3學分 | 陳述句 | 不要必修 | 模糊 |
| F複合 | 長句 | 多個排除 | 半精確 |
| C師型 | 口語 | 無 | 精確 |
... 以此類推覆蓋全部組合

---

## 6. 特殊類型（需特別生成）

### 6a. 錯誤資訊型
使用者有錯誤假設：
```
林建秀老師不是有開國際金融嗎（老師名字記錯）
政治系大一可以修法律系選修嗎（身分判斷不確定）
```

### 6b. 比較型
```
投資學和財管哪個比較好修
財報分析和個體經濟哪個對考研比較有幫助
```

### 6c. 「幫我推」型
```
幫我推薦幾門通識課
有什麼課適合文組學生修
有沒有適合大三修的進階資料科學課
```

### 6d. 時事/流行型
```
有沒有 AI 相關的課
ChatGPT 時代有沒有相關課
有沒有跟 ESG 永續有關的課
```
